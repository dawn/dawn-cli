Dawn-CLI
========
CLI for [Dawn](https://github.com/dawn/dawn)

If you are looking for the ruby API
[Dawn-API](https://github.com/dawn/dawn-api)

## Installation
```shell
gem install dawn-cli
```

## Usage
```shell
dawn
```
Help is printed if no commands are given.
